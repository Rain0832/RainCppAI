快速开始

  1. 配置 API Key — 在上方 API Key 标签页中填入对应模型的密钥，保存后即可使用该模型。
  2. AI 聊天 — 点击「AI 聊天助手」，创建新对话或选择历史会话，在底部输入框提问。支持 Enter 发送，Shift+Enter 换行。
  3. 切换模型 — 在聊天页顶栏的下拉框选择不同模型（需先配置对应 API Key）。
  4. 主题切换 — 点击右上角 ☀️/🌙 按钮切换亮色/暗色模式，设置会自动保存。
提示：API Key 仅保存在浏览器本地存储中，不会上传到服务器。切换浏览器或清除数据后需重新配置。
