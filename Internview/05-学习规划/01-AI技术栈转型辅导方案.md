## 二、AI 核心知识速通

### 2.1 LLM = 超大文本预测函数

```cpp
Token predict(const vector<Token>& context) {
    vector<float> embeddings = tokenizer.encode(context);
    for (auto& layer : transformer_layers) {
        embeddings = layer.forward(embeddings);
    }
    return sample(softmax(embeddings));
}
```

### 2.2 Transformer / Attention

- Q(Query) = 当前 token 在问 "关注上文哪些信息"
- K(Key) = 上下文每个 token 的标签
- V(Value) = 上下文每个 token 的内容
- `score = softmax(Q × K^T / √d)`, `output = score × V`

### 2.3 Agent（ReAct 模式）

`Thought → Action → Observation → Thought → ... → Answer`

### 2.4 Prompt Engineering 核心技巧

System Prompt / Few-shot / Chain-of-Thought / Output Format / Context Engineering
